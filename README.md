
# Oivan Tech Task Demo

## Overview
This is a simple demonstration project for the Oivan tech task.

## Features
- Clean project structure
- Easy to understand and extend
- Well-documented code

## Getting Started

### Prerequisites
- Flutter SDK
- Dart

### Installation
```bash
flutter pub get
```

### Running the Demo
```bash
flutter run
```
